Frontend Documentation
=====================

This section covers the Flutter frontend of AeraSync.

Architecture
-----------

The frontend is built with Flutter, providing a cross-platform application experience
for web, iOS and Android. It communicates with the backend API to perform aerator comparisons
and display results to the user.

State Management
--------------

The application uses a state management approach to handle data flow between the UI and business logic.
This ensures a clean separation of concerns and an efficient update mechanism for the UI.

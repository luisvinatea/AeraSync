AeraSync Documentation
=====================

Welcome to AeraSync's documentation. This documentation covers both the backend Python API and frontend Flutter application.

Overview
--------

AeraSync is a tool for comparing aerator performance and efficiency. It provides users with data analysis capabilities 
through a Flutter frontend and Python backend API.

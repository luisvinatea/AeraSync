Flutter Application Reference
--------------------------

This section provides documentation for the AeraSync Flutter frontend application.

Architecture
^^^^^^^^^^

The Flutter application follows a clean architecture pattern with the following layers:

* **Presentation Layer**: UI components and state management
* **Domain Layer**: Business logic and models
* **Data Layer**: Data sources and repositories

Pages

The application consists of several key pages:

* **Results Page**: For displaying comparison results
